Problem Statement

A program is required to prompt the terminal operator for the maximum and minimum temperature readings on a particular day, accept those readings as integers, and calculate the average temperature, calculated by (maximum temperature + minimum temperature)/2.The program should display the maximum temperature, the minimum temperature, and the average temperature.

5. JavaScript Code

Write the JavaScript code based on everything you've done.

Here's the code:


// Find average temperature

// Declare variables
var averageTemperature = 0;
var maximumTemperature = 0;
var minimumTemperature = 0;
var output = "";

// Prompt the user for temperatures
maximumTemperature = prompt("Enter the maximum temperature");
minimumTemperature = prompt("Enter the minimum temperature");

// Convert to numbers
maximumTemperature = Number(maximumTemperature);
minimumTemperature = Number(minimumTemperature);

// Calculate the average
averageTemperature = (maximumTemperature + minimumTemperature) / 2;

// Display the results
output = output + "The maximum temperature is " + maximumTemperature + "\n";
output = output + "The minimum temperature is " + minimumTemperature + "\n";
output = output + "The average temperature is " + averageTemperature + "\n";

document.write(output);
