Problem Statement

A program is required to read three numbers, add them together and print their total in the following foramt::

The total of 1 + 2 + 3 = 6.

1. Nouns and Verbs

Identify the nouns and verbs in the problem statement.

PROBLEM STATEMENT: A program is required to read three numbers, add
them together and print their total.

Nouns: three numbers, total

Verbs: read, add, print

2. Defining Diagram

Convert the nouns and verbs into variable names and processes and create a defining diagram.

INPUTS:
  number1
  number2
  number3

PROCESSING:
  read three numbers
  add numbers together
  print total number formatted as "The total of 1 + 2 + 3 is 6."

OUTPUTS:
  total number formatted as "The total of 1 + 2 + 3 is 6."

3. Solution Algorithm

Create a solution algorithm based on the defining diagram.

AddThreeNumbers
  Initialize number1 to 0
  Initialize number2 to 0
  Initialize number3 to 0
  Initialize total to 0

  Prompt for number1 with "Enter the first number:"
  Prompt for number2 with "Enter the second number:"
  Prompt for number3 with "Enter the third number:"

  Convert number1 to a number
  Convert number2 to a number
  Convert number3 to a number

  Set total = number1 + number2 + number3

  Display "The total of " + number1 + " + " +
          number2 + " + " + number3" +
          " is " + total + "."
END

4. Checking the Solution Algorithm

Create a test plan for the algorithm. Remember to use this to discover if your algorithm needs work!

VARIABLES: number1, number2, number3, total

TEST CASE 1:

  INPUT VALUES: 10, 20, 30

  EXPECTED RESULT: "The total of 10 + 20 + 30 is 60."

  ACTUAL RESULT: "The total of 10 + 20 + 30 is 60."

TEST CASE 2:

  INPUT VALUES: 40, 41, 42

  EXPECTED RESULT: "The total of 40 + 41 + 42 is 123."

  ACTUAL RESULT: "The total of 40 + 41 + 42 is 123."

TEST CASE 3

  INPUT VALUES: "A", "B", "C"

  EXPECTED RESULT: "The total of NaN + NaN + NaN is NaN."

  ACTUAL RESULT: "The total of NaN + NaN + NaN is NaN."

TEST CASE 4

  INPUT VALUES: "A", 2, 3

  EXPECTED RESULT: "The total of NaN + 2 + 3 is NaN."

  ACTUAL RESULT: "The total of NaN + 2 + 3 is NaN."

5. JavaScript Code

Write the JavaScript code based on everything you've done.

  • Here's the code:

// Add three numbers

//Declare the variables
var number1 = 0;
var number2 = 0;
var number3 = 0;

// we'll need the total later, but we declare it here.
var total = 0;

// Get the numbers from the user
number1 = prompt("Enter the first number:", "1");
number2 = prompt("Enter the second number:", "2");
number3 = prompt("Enter the third and last number:", "3");

// Convert the inputed strings to numbers
number1 = Number(number1);
number2 = Number(number2);
number3 = Number(number3);

// Calculate the total
total = number1 + number2 + number3;

// Display the total. You can split the line of code at a + sign
document.write("The total of " + number1 + " + " + number2 + " + " +
               number3 + " is " + total + ".");
