Unit 3: Program Design and Debugging

Welcome to Learning Unit 3 - Program Design and Debugging!

We've done some work with pseudocode before, but in this learning unit we're going to dive in and see exactly what we do when we have a problem we want to solve with code. You'll learn how to break that problem down, design the solution algorithm, and then turn that into runnable code.

And as you'll soon learn, JavaScript code often doesn't run correctly the first time you try it. There are many things that can go wrong. As you write JavaScript code you may spell something incorrectly, forget a semi-colon, or make any number of syntax errors. And sometimes finding those errors requires some hard work! Finding and fixing errors is called debugging! And, since debugging is a huge part of any programmers job, we need to take a look at how it's done.
