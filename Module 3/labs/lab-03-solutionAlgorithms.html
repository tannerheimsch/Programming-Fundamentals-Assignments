My Turn:

Problem Statement: A program is required to prompt the user for a Fahrenheit temperature,
convert it to Celsius and print the converted temperature. 
Celsius is determined by subtracting 32 from the Fahrenheit 
temperature and multiplying the result by 5/9.

SOLUTION ALGORITHM:

ConvertFahrenheitToCelsius
  prompt for temperature in fahrenheit
  celsius is equal to (5 divided by 9) times (fahrenheit minus 32)
  display celsius temperature
END

Your Turn:

YOUR SOLUTION ALGORITHMS START AFTER THIS LINE
1. Problem Statement: A program is required to prompt the user for  a distance in miles,
calculate the distance in kilometers, and print the kilometers. One mile 
is 1.6 kilometers.

SOLUTION ALGORITHM:

ConvertMilesToKilometers
  Initialize distanceInMiles to 0
  Initialize distanceInKilometers to 0

  Prompt for distanceInMiles with "Enter distance in miles"

  Set distanceInKilometers = distanceInMiles times 1.6

  Display "Distance in kilometers: " + distanceInKilometers
END

2. Problem Statement: A program is required to prompt a user for 
radius and height of a cylinder. The program will calculate the 
volume of the cylinder and output the results. The formula for 
the calculation is:
  
  - Volume: PI times the radius squared times the height.

SOLUTION ALGORITHM:
CalculateCylinderVolume
  Initialize radius to 0
  Initialize height to 0
  Initialize volume to 0
  Initialize pi to 3.14

  Prompt user for radius with "Enter the radius of the cylinder"
  Prompt user for height with "Enter the height of the cylinder"

  Set volume = pi times (radius squared) times height

  Display "The volume of the cylinder is " + volume
END
