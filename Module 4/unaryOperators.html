Unary Operators

Unary operators offer "short cuts" when we are writing code. They seems a little strange at first. But, after you've practiced with them, you'll find them easy and quick to use. They are really handy and programmers use them all the time. Any trick to save typing and time makes a programmer more productive!

The Unary Operators

Operator Function
++ Increments a Number
-- Decrements a Number
typeof Returns a string representing the data type

Using the Unary Operators

Here's the first way we've seen to add one to a number. The value in myCount is set to 5 in code line 02. Then in line 06, 1 is added to the 5 to make 6...and the 6 replaces the 5 as the new value in myCount.


// Addition with the +
var myCount = 5;

document.write("myCount: " + myCount + "\n");

myCount = myCount + 1;

document.write("myCount: " + myCount);

But that is too much typing! There is a shortcut we can use...

"Unary" actually means that there is only one variable or value involved with the operator. Usually we have to have something on either side of the operator, like the code statement below. 


myCount = myCount + 1;

But, a unary operator has just one. This code is SHORTER and does the same thing. (The comment just reminds us that we're adding 1, just like we did in the previous satement.)


myCount++; // Increment, or add one, to myCount


// Incrementing with ++
var myCount = 5;

document.write("myCount: " + myCount + "\n");

myCount++;

document.write("myCount: " + myCount);

I bet you can guess what this one does!


myCount--; // ???

That's right, it subtracts one from myCount


// Decrementing with --
var myCount = 5;

document.write("myCount: " + myCount + "\n");

myCount--;

document.write("myCount: " + myCount);

We're only going to look at one other unary operator: typeof(). It looks like a call to a function with parentheses. It tells us what type of data we have. It actually returns the strings "number", "string ", and "boolean ".


    typeof(5) <-- this will be "number"

The typeof() operator is useful for programmers when they are attempting to find out what TYPE of data is currently stored in a variable. But, here are some sample lines of code to help you understand how it works.


// Demo of typeof()
var myName = "Homer";
document.write("The type of the myName variable is:\"" + 
                                            typeof(myName) + "\"");
document.write("\n");    
document.write("The type of 10 is: \"" + typeof(10) + "\"");
document.write("\n");
document.write("The type of \"10\" is: \"" + typeof("10") + "\"");
document.write("\n");
document.write("The type of (5 == 5) is: \"" + typeof(5 == 5) + "\"");
