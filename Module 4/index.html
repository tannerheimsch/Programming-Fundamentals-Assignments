Unit 4: Operators and Functions

Welcome to Learning Unit 4 - Operators and Functions

Unit Learning Activities

In this unit we will explore how data is used and manipulated by a program. We know, for example, that pieces of data can be held in variables while the program uses them. We "assign" a value to a variable. But, to be a programmer we need to know more about how it all works!

Let's take a closer look at the "operators" we can use in programming to manipulate data. After all, that is the goal of most programs - to input data, to process that data, and to output information that can be used for decision-making. Here's an example of what we'll be practicing in this unit:

  • Use certain operators to do math (add, subtract, divide, multiply).
  • Use operators to shortcut math statements.
  • Use operators to compare values in variables (equal, greater than, less than, etc.).
  • And more!

Then we'll explore functions, which let us break our code up into smaller, reusable pieces. Up until now, you've written a single block of code. But functions are like little smaller programs that we use to separate complex processing steps so we can reuse them later. You'll work with simple functions in this unit and work with more complex ones later.
