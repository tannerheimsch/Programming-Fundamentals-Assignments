Operators That Add and Subtract

You have been adding and subtracting numbers since grade school. It seems just logical that you would need to be able to add and subtract in JavaScript code, doesn't it? That's what this part of the unit is all about...the operators used to add and subtract. They may seem simple at first but there are a few surprises!

The Additive Operators

In math, additive refers to both addition and subtraction. You can think of subtraction as just adding with negative numbers.

  • The plus sign (+) is used for math addition and string concatenation.
  • The minus sign (-) is used for math subtraction.

Examples of Using the Additive Operators

  • Here's an example of Adding and Subtracting Directly into Variables.
  • Notice line 04 in the code example below. It's adding two numbers, and assigning the result of that calculation (the sum) directly into the totalCount variable. So, at the end of line 04, the value in totalCount is 766.
  • Also, look at line 09, which is subtracting 200 from the value assigned to the totalCount variable (back in line 04). When line 09 in the code is completed, the result of that calculation (566) now replaces the value previously assigned to the totalCount variable back in line 04.

// Additive operators into variables
var totalCount;

totalCount = 223 + 543;

document.write("totalCount: " + totalCount);
document.write("\n");

totalCount = totalCount - 200;

document.write("New totalCount: " + totalCount);

Here are more examples of Adding and Subtracting with Variables.

  • Notice line 07 below. JavaScript knows that the value assigned to the highPrice variable is 1000.99 (see line 04). And, JavaScript knows that the lowPrice variable has been assigned a value of 0.99 (line 03).
  • Therefore, when we tell JavaScript that totalPrice should be the result of lowPrice + highPrice, it knows that we mean 1000.99 + 0.99.
  • NOTE: If we made a mistake in our code, and FORGOT to assign a value to a variable, JavaScript assumes its value is 0 (ZERO)! Another situation where a variable may have a value of zero...if you SPELL the variable name incorrectly in the var statement or in the assignment statement!
// Additive operators with variables
var differencePrice = 0;
var highPrice = 1000.99;
var lowPrice = 0.99;
var mediumPrice = 199.95;
var totalPrice = 0;

totalPrice = lowPrice + highPrice;
differencePrice = highPrice - mediumPrice;

document.write("totalPrice: " + totalPrice + "\n");
document.write("differencePrice: $" + differencePrice);

Let's review the Concatenation of strings. It may seem like this is a strange place to do that. But, it's the PERFECT place!

As you recall, Concatenation is where we add strings together. Like adding a first name together with a last name, along with a space in betwen. Or a name with an age. We often do this for building up output like text or HTML. Or to format money.

The plus sign (+) is used to add numbers together. But, it is also used for concatenation, which is the process of combining strings with other strings (or numbers) to form a new longer string. In fact, you'll notice in the code below, that the <p> tags can be concatenated too.

This is where a variable's data type becomes SUPER important!

  • If you use the plus sign with number variables, it adds them together.
  • If you use the plus sign, and one or more of your variables is a string data type, it concatenates!!!

In the example that follows, the firstName and lastName variables are both strings.

  • Line 06 in the code is concatenating three strings. It's taking the value of the firstName variable (which is "Fred") and placing next to it a string that contains only one space (looks like this: " "). After the space it's placing the value of the lastName variable ("Flintstone"). Then the new string, "Fred Flintstone" is assigned to the variable called fullName.
  • Hence, the result is this string: Fred Flintstone , which you can see displayed as a result of the document.write command on line 10.

// Concatenation with +
var firstName;
var fullName;
var lastName;

firstName = "Homer";
lastName = "Simpson";

// put a space between the first and last name.
fullName = firstName + " " + lastName;

document.write("<p>firstName: " + firstName + "</p>");
document.write("<p>lastName: " + lastName + "</p>");
document.write("<p>The fullName: " + fullName + "</p>");

Let's look at examples of concatentation with the variables are both strings and numbers, mixed together.

  • When the plus sign has one string to its left OR right and a number on the other side, the number will be converted to a string and concatenated to the string.
  • In the code below, petWeight is a number variable. But, since it's being concatenated with other strings in the document.write statement, JavaScript will NOT try to do addition, even though petWeight is a number. Rather, it will just concatenate the 10 as directed.

// Concatenation with numbers
var petName = "Santa's Little Helper";
var petWeight = 10;

document.write("My pet, " + petName + ", weighs "
        + petWeight + " lbs.");

How about an example of concatenation when you don't expect it!.

When you have a string that contains just numbers and you try to add it to a true number JavaScript will concatenate it and not do addition.

  • In line 5 below, JavaScript inspects the value of invoiceTwo and immediately determines that the invoiceTwo variable is a STRING datatype, even though it has the number 599 inside. However, that's not the number 599. It's the characters "5", "9", and "9" put togehter. Looks can be deceiving!
  • In the output, you would have expected the number 1599 if addition had been performed successfully. But, instead the 599 was CONCATENATED (or placed next to) the 1000...so the result is 1000599.

// Whoops!
var accountTotal;
var invoiceOne = 1000;
var invoiceTwo = "599";

accountTotal = invoiceOne + invoiceTwo;

document.write("Account Total: " + accountTotal + " <-- ???");

Converting from a string to a number.

When you have string that contains a valid number, and you want to use the number in a calculation, you CAN convert it to a number by using the Number() function. We've done this a few times already.

  • The numberString variable in the code below is assigned value of 56.999 as a string (see the quotes?). Because of that, the total variable in line 06 is assigned the concatenation of 67.884 and 56.999. The document.write statement in line 08 displays the result: 67.88456.999. No addition happening there!!!

// Converting a string to a number
var numberString = "56.999";
var number = 67.884;
var total;

total = number + numberString;

document.write("total: " + total + " <-- String concatenation");
document.write("\n");

total = number + Number(numberString);

document.write("total: " + total + " <-- That's better!");
