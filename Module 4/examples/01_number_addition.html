SOLUTION ALGORITHM:

AddTwoDecimalNumbers

  initialize number1 to 0.0
  initialize number2 to 0.0
  initialize sum to 0

  prompt for number1 with "What is the first decimal number?"
  prompt for number2 with "What is the second decimal number?"

  convert number1 to a number
  convert number2 to a number

  number1 = number1 * 100
  number2 = number2 * 100

  sum = number1 + number 2

  sum = sum / 100

  Display "The sum is " + sum

END

Explanation of Algorithm

Note this section of the pseudocode:

  number1 = number1 * 100
  number2 = number2 * 100

  sum = number1 + number 2

  sum = sum / 100

Because decimals can't be added together with precision, we are multiplying the numbers by 100 to shift the decimal. This will help us preserve some precision.

When we get the sum, we then divide by 100 to move the decimal back.
