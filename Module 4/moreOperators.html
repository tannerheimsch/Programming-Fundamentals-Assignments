More Assignment Operators

Let's look at a few miscellaneous operators that are useful. We call them Compound Assignment Operators. We use them a lot when altering a variable's value mathematically.

Compound Assignment Operators

Operator Function
+= Add to a variable
-= Subtract to a variable
*= Multiply into a variable
/= Divide into a variable

Using Compound Assignment Operators

Adding into a variable (+= )

You probably already know that we can add a number to a variable like this, using in-place assignment:


// The old way...Adding with +
var myNumber = 0;
myNumber = myNumber + 10; //increment myNumber by 10
document.write("myNumber: " + myNumber);

But we can also use a shortcut Compound Assignment Operator too. Since this is less typing we have fewer chances of making an error. Also, this is the way you will see professional programmers do it. 


// The new way...Adding with +=
var myNumber = 0;
myNumber += 10; //increment myNumber by 10 with Compound Assignment Operator
document.write("myNumber: " + myNumber);

Subtracting intro a variable (-= )

Of course, we can subtract a number from a variable like this:


// Subtracting with -
var myNumber = 100;
myNumber = myNumber - 10;
document.write("myNumber: " + myNumber);

And we can use a shortcut Compound Assignment Operator too. Since this is less typing we have fewer chances of making an error. Just like with additiong, this is the way you will see professional programmers do it.


// Subtracting with -=
var myNumber = 100;
myNumber -= 10; //decrement the value of myNumber by 10
document.write("myNumber: " + myNumber);

Multiplying into a variable (*= )

As you might now guess, we can multiply into a variable by a number, and store the answer back in the original variable. It looks like this:


// Multiplying with *
var myNumber = 100;
myNumber = myNumber * 2;
document.write("myNumber: " + myNumber);

This works too:


// Multiplying with *=
var myNumber = 100;
myNumber *= 2;
document.write("myNumber: " + myNumber);

Dividing into a variable (/= )

I think you could figure out that this one was coming! Dividing works like this:


// Dividing with /
var myNumber = 100;
myNumber = myNumber / 2;
document.write("myNumber: " + myNumber);

But, dividing like this will give us the same results:


// Dividing with /=
var myNumber = 100;
myNumber /= 2;
document.write("myNumber: " + myNumber);
