My Turn:

A customer deposits 2 checks and wants a receipt printed for the entire amount of the deposit. If the entire deposit is greater than $1000.00 then the customer earns a $5.00 bonus. The receipt should display the customer's name and total deposit. If the customer has a bonus the receipt should include the $5.00 in the total deposit and alert the customer.

Here's my pseudocode.

Customer Deposit Receipt Pseudocode

BEGIN
  initialize check1 to 0
  initialize check2 to 0
  initialize customerName to ""
  initialize output to ""

  prompt for customerName
  prompt for check1 with "What is the first check?"
  prompt for check2 with "What is the second check?"

  totalDeposit = check1 + check2

  output = customerName + "\n"
  output = output +  totalDeposit + "\n"

  IF totalDeposit > 1000 THEN
    totalDeposit = totalDeposit + 5
    output = output +  "Congratulations - you earned a $5.00 bonus!"
  ENDIF

  display output
END

Pay attention to all the things I did in my example. Look at the indentation. Look at how specific I was. As you write pseudocode, strive for that level of clarity.

Your Scenario:

An instructor needs to key in three grades. The grades need to be averaged. Then the letter grade should be determined. An average score between 70 and 100 earns a grade of "P", whereas any average score below 70 earns an "F". The average and the letter grade should display on a screen. 
YOUR PSEUDOCODE STARTS AFTER THIS LINE:
BEGIN
  // Define variables
  initialize grade1 to 0
  initialize grade2 to 0
  initialize grade3 to 0
  initialize average to 0
  initialize letterGrade to ""

  // Prompt user for grades
  prompt for grade1 with "Enter grade 1:"
  prompt for grade2 with "Enter grade 2:"
  prompt for grade3 with "Enter grade 3:"

  // Calculate the average of the three grades
  average = (grade1 + grade2 + grade3) / 3

  // Define output concatenation
  output = average + "\n"
  output = output +  letterGrade

  // Determine the letter grade based on the average score
  IF average >= 70 AND average <= 100 THEN
    letterGrade = "P"
  ELSE
    letterGrade = "F"
  ENDIF

  // Output
 display output
END
