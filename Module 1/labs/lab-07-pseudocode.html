Lab Steps

  1. Open this file in both Firefox and Notepad++.
  2. Add your name to the page.
  3. Test that you added your information correctly.
  4. Write pseudocode for the following procedure:
    1. For every employee in the company
      • Ask for his or her last name and store it in a variable.
      • Ask for his or her weekly salary and store it in a variable.
        • If the salary is not greater than 1000, display a message that says “Evaluate employee name for a raise.” (where name is the name of the employee you stored.)
        • Also, increment a variable that keeps track of the number of employees that need evaluation.
    2. At the end of the program, after all of the employees have been scanned, display a final message that shows the total number of employees that need to have their salaries evaluated. For example, if there were 100 employees, but only 20 employees that needed evaluation, the program should say:

      “The total number of employees that need to be evaluated is 20.”

    HINT: You will need a WHILE statement and inside the WHILE you will have one IF statement.

    Think about the keywords you'd use for each step and include those, but remember that pseudocode isn't a real programming language - it's just a loose way of expressing how your program might work.

    Pay special attention to the algorithm. What pseudocode keywords do you use to compare and make a choice? What about asking for input and storing values?

  5. Add your pseudocode so it appears in the black box, after the line: YOUR PSEUDOCODE STARTS AFTER THIS LINE:
  6. Save your lab and take some extra time to try to run through the logic in your head.

Your Pseudocode:

YOUR PSEUDOCODE STARTS AFTER THIS LINE:
BEGIN
  // Define variables
  initialize numEmployees to 0
  initialize numEvaluted to 0
  initialize salary to 0
  initialize lastName to ""

  // Prompt user for information, evaluate whether or not they need a raise
  WHILE thereAreMoreEmployees DO
  prompt for lastName with "Enter your last name:"
  prompt for salary with "Enter your weekly salary:"
    IF salary <= 1000 THEN
      display "Evaluate employee " + lastName + " for a raise."
      numEvaluated = numEvaluated + 1
    ENDIF
    set numEmployees = numEmployees + 1
  ENDWHILE
  
  // Display total # of employees that need a salary evaluation
  display "The total number of employees that need to be evaluated is " + numEvaluated + "out of " + numEmployees + "total employees."
END
END YOUR PSEUDOCODE BEFORE THIS LINE
