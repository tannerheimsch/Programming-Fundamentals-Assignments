Nested loops

File: unit6/nestedLoops.html

Loops Within Loops, oh my.

This is a powerful concept that is crucial for most modern programming. We'll start simply, though.

Sometimes a problem statement requires nested loops. Here's a Solution Algorithm that shows one.

CreateGridOfXs
  declare two variables for loop counters
  set the outer loop counter to zero
  WHILE outer loop counter is less than 5
    set the inner loop counter to zero
    WHILE inner loop counter is less than 5
      Output an "X" to the page
      Increment the inner loop counter
    ENDWHILE
    Output a line break to the page
    Increment the outer loop counter
  ENDWHILE
END

Any code can go inside a loop, even another loop. This gives us some interesting possibilities.


/*
  This is an example of nested looping 
  with while loops.
*/
function firstNestedLoops() {
    
  // Declare variables
  var innerCounter;
  var outerCounter;
  
  // Outer loop
  // Increment its counter first
  outerCounter = 0;
  while (outerCounter < 5) {        
      
    // Inner loop
    // Increment its counter first
    innerCounter = 0;
    while (innerCounter < 5) {
      document.write("X");
      innerCounter++;
    }
    document.write("<br />");
    outerCounter++;
  }
}

The Concept

Nested Loops (Nested Loops (Nested Loops (Nested Loops)))

Here's how this works. We need a loop counting variable for each of the while statements in our program. In this case that's two. There's no need to assign them a value yet.

// Declare Variables

var innerCounter;
var outerCounter;

Next, we create the outer loop, when means the loop that runs first. We have to make sure we assign the starting value to the counter just before the loop starts.

// Outer loop

outerCounter = 0;
while (outerCounter < 5) {
  // This will run five times
  ...
  outerCounter++;  // <-- Increment the outer loop counter.
}

This outer loop will run five times and all code in it will run five times, even another loop!

Now we'll put another loop inside the first loop. Assign the loop counter for this loop just before the loop starts. It has to be just before the loop so it will be reset to zero every time the loop starts.

// Adding the inner loop

outerCounter = 0;
while (outerCounter < 5) {
  // This will run five times
  
  // This is a nested loop that is 
  // completely inside the first loop.
  innerCounter = 0;
  while (innerCounter < 5) {
    // This will run five times for 
    // every time the outer loop runs,
    // or 25 times.
    ...
    innerCounter++;
  }
    
  outerCounter++;  // <-- Increment the inner loop counter.
}

Now let's do some work! We'll just do something simple like writing an "X" to the page. We only put this in the inner loop so it will run every time.


// The full program

outerCounter = 0;
while (outerCounter < 5) {
  // This will run five times
  
  // This is a nested loop that is 
  // completely inside the first loop.
  innerCounter = 0;
  while (innerCounter < 5) {
    // This will run five times for 
    // every time the outer loop runs,
    // or 25 times.
    document.writing("X");  // <-- 25 times!
    innerCounter++;
  }
  document.write("<br />");  // <-- Go to the next line.
  outerCounter++;  // <-- Increment the inner loop counter.
}
