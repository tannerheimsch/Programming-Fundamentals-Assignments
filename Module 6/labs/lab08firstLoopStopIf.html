SOLUTION ALGORITHM GOES HERE:

TotalNonZeroNumbers
  Initialize userNumber to 1
  Initialize userNumberSum to 0

  WHILE userNumber is not equal to 0
    prompt for and convert userNumber with "Please enter a number greater than 0 (or type 0 to quit):"
    IF userNumber is not a number THEN
      alert "Error: Please enter a valid number"
      continue loop
    ENDIF
    display "Number entered: " + userNumber + "\n"
    IF userNumber is greater than 0 THEN
      add userNumber to userNumberSum
    ENDIF
  ENDWHILE

  Display "Total of all numbers entered: " + userNumberSum + "\n"
  Display "Good bye!"
END
