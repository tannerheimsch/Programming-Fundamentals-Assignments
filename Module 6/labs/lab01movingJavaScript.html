Lab Steps

  1. Open this file in Firefox and Notepad++.
  2. Add your name to this file in the appropriate spot.
  3. Open Notepad++ and create a new empty file.
  4. Save this new file into the unit6/labs/js directory.
  5. Give the file the name lab01movingJavaScript.js. It must be spelled EXACTLY like that or this lab won't work!
  6. Open this Lab 1 HTML file lab01movingJavaScript.html in Notepad++.
  7. Scroll down to the JavaScript code, it's in the place we usually key in our JavaScript code.
  8. Copy the JavaScript code, and just the JavaScript code, and paste it into the file you made named lab01movingJavaScript.js. Remember, don't take the <script> tags, just take the JavaScript code.
  9. Add this line above the code you just pasted - it should be the first line of the file:
    function lab01movingJavaScript() {
  10. Add this after the end - it should be the last line of the file.
    }
  11. Now we need to indent our code. The function line and the final } should be all the way over to the left. Our code between those lines must then be indented 2 spaces.

    
function lab01movingJavaScript() {
      //your code
  // should all be indented
  // two spaces over
}
  12. Finally, add the function call. The function call looks like this:
    lab01movingJavaScript();
  13. Save that file. We're done with it.
  14. Now edit the lab HTML file (this file you're reading) and delete everything between the <script> tag and the </script> tag. That's right, we don't need the JavaScript code any more now that we've put the code to another file.
  15. Finally, since the <script> tags are empty, we can remove those too.
  16. Refresh this page. Did it work? Did you still get the output? (If you didn't, go through each of the steps above to see where your error is. Check each step. Is everything is spelled correctly? Has it been keyed into the proper location?)

How did that work? Well, if you look at the source of this page, you'll find this line in the output section right below the <script> tags you just removed:


<script src="js/lab01movingJavaScript.js"></script>

This line loads the lab01movingJavaScript.js file that you created to the document. Without this line, the functions you placed in the lab01movingJavaScript.js file wouldn't be visible to the page and you'd get errors.
