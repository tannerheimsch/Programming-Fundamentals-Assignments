Design an algorithm that will output the seven times table from 1 through 20. Each output line will show the calculation like this:

  • 7 × 1 = 7
  • 7 × 2 = 14
  • etc.

displaySevenTimesTable
  Initialize factor to 7
  Initialize count to 1

  WHILE count <= 20
    SET product = factor * count

  ENDWHILE
