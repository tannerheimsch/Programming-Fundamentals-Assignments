Problem Statement


A program is required that asks a user to enter a number for a "counter". The program then asks the user to enter numbers, one at a time, in a loop for the number of times specified in the "counter". The program will output the "counter" number, each entered number, and after all the numbers are entered, then output the average of the entered numbers.

3. Solution Algorithm

This is the solution algorithm. Replace the ??????? in the pseudocode below to complete the algorithm. 

calculateAverage
  prompt user for loopCounter
  convert loopCounter to a number
  LOOP starting at 0 to loopCounter in increments of 1
    prompt user for numberIn
    convert numberIn to a number
    add numberIn to numberTotal
    output loopCounter and numberIn
  END LOOP
  numberAverage = numberTotal / loopCounter
  output numberTotal and numberAverage
END
