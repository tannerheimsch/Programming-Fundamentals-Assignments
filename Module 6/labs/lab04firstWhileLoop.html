Lab Problem Statement

Write a program that uses a WHILE loop. Inside the loop, the program asks a user to enter something, displaying their input back to them each time. When the loop has completed five times, thank the user for participating.

Use a while loop with a counter variable that you increment with the ++ operator. The while loop should check the counter to know when to stop. Ensure the loop only happens five times!

SOLUTION ALGORITHM GOES HERE:

UserInputWhileLoop
  Initialize userInput to ""
  Initialize userGratitude to ""
  Initialize loopCount to 0

  WHILE loopCount < 5
    prompt user for userInput with "Please enter something you would like to see displayed:"
    display "You entered: " + userInput + "\n"
    add 1 to loopCount
  ENDWHILE

  userGratitude = "Thanks for participating!"
  display userGratitude
END
