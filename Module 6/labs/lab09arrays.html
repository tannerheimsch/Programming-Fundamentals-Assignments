Lab Problem Statement

Write a program that asks for student usernames. Collect student usernames until the user enters "done". Once all the data has been gathered, the program should display the total number of records you collected. It should then convert the usernames to student email addresses (by appending @student.cvtc.edu") and display the converted addresses in a list as shown in the example output.

You'll use both FOR and WHILE loops in this problem, as well as an IF statement. You'll use an array to store the usernames.

SOLUTION ALGORITHM GOES HERE:

StudentEmailGenerator
  Initialize array studentUsernameList to ""
  Initialize array studentEmailList to ""
  Initialize generatorDone to "false"
  Initialize totalRecords to 0

  WHILE generatorDone is false
    prompt for studentUsername with "Enter a student username (i.e. jsmith), type 'done' once you are finished."
    IF studentUsername is "done"
      IF studentUsernameList = 0 THEN
        alert "Please enter at least one username."
      ELSE
      generatorDone = "true"
      ENDIF
    ELSE IF studentUsername includes a space THEN
      alert "Username cannot contain spaces. Please try again."
    ELSE studentUsername is not an empty string THEN
      push studentUsername to studentUsernameList
    ENDIF
  ENDWHILE

  FOR each studentUsername in studentEmailList
    SET studentEmail = studentUsernameList + "@student.cvtc.edu"
    push studentEmail to studentEmailList
  END FOR

  Set totalRecords = length of studentUsernameList

  IF totalRecords = 1 THEN
    display "You entered 1 username. Here is the email address:\n\n"
  ELSE
    display "You entered " + totalRecords + " usernames. Here are the email addresses:\n\n"
  ENDIF
  display studentEmailList + "\n"
END
