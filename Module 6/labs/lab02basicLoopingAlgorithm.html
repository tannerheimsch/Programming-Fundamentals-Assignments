Problem Statement

Define an algorithm that asks a user to enter 5 strings using a loop. Output each string and its length as it is entered, and output the calculated average length of the strings after the entry has finished.

1. Nouns and Verbs

I've already identified the nouns and verbs in the problem statement for you, but please study it carefully.

PROBLEM STATEMENT:
  Define an algorithm that ask a user to enter 5 strings
  using a loop. Display each string as they are entered
  and display the calculated average length of the strings after
  the entry has finished.

Nouns: 5 strings, 5 lengths, average length

Verbs: ask, calculate, display

2. Defining Diagram

I've already converted the nouns and verbs into variable names and processes and created a defining diagram for you. Study this diagram though and be sure you understand how it works.

INPUTS:
  stringIn (five times)

PROCESSING:
  prompt user 5 times for a string
  display each string
  determine and display length of each string
  calculate the average length of the 5 strings
  display average length

OUTPUTS:
  stringIn (five times)
  length of stringIn (five times)
  average length

3. Solution Algorithm

This is the solution algorithm. Replace the ??????? in the pseudocode below to complete the algorithm

calculateAverageStringLength
  set counter to zero
  WHILE counter < 5
    prompt user for stringIn
    determine length of stringIn
    output stringIn and its length
    add stringIn length to totalLengths
    add 1 to counter
  ENDWHILE
  calculate avgLengthString = totalLengths/5
  output "The average length of the entered strings is " + avgLengthString + "."
END
