Lab Problem Statement:

Write a program that asks a user to enter a number. Use zero as a
"sentinel" value. In other words, compare the number to zero in the loop condition. If
the number entered by the user is not equal to zero, then keep going.
Ask the user to enter another number, add it to the
original number, and store the result as a total. As long as the number
entered by the user is not 0, continue asking the user to enter
another number, and add it to the total. Once the user enters a zero
then it's time to end the loop. As you progress through the loop, display each of
the numbers entered by the user. After the loop display the cumulative total and display the sign off
message "Good bye!". Use the example output as your guide.

4. JavaScript Code

function lab09unknowLoopCount() {
  var cumulativeTotal = 0;
  var message = "Good bye!";
  var numberIn;
 
  while (numberIn !== "0") {
    numberIn = prompt("Enter a number (0 to quit).");
    document.write("Number entered:  " + numberIn + "
");
    cumulativeTotal += Number(numberIn);
  }
 
  document.write("Total of all numbers entered:  " + cumulativeTotal);
  document.write("
");
  document.write(message);
}
