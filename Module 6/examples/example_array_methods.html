Lab Problem Statement

Write a program that prompts the user for a single word. If the word entered is a palindrome (a word that is spelled the same forwards and backwards), then state that the word is a palindrome. If it's not, then state that the word is not a palindrome.

Implement this program using a function you define called isPalindrome() that accepts the word as its argument and returns true or false.

Your function should not prompt for input or display any output to the screen. The function should simply take in the word, test it, and return either true or false. You will then use an IF statement to test the result of the function to determine which message to display.

Hint: Convert the string to an array of characters using .split(""). Then use the .reverse() function on the Array to reverse the letters in the array.

3. Solution Algorithm

Create a solution algorithm based on the defining diagram.

        
function isPalindrome(userWord)
  userWordArray = userWord.split("")
  userWordArrayReverse = userWord.split("")
  result = userWordArray.join("") === userWordArrayReverse.join("")
END function

checkPalindrome
  prompt for userWord
  if isPalindrome(userWord) then
    message = "The word " + userWord + " is a palindrome."
  else
    message = "The word " + userWord + " is not a palindrome."
  endif
  display message
END
