Unit 2: JavaScript Syntax and Variables

Example: String Variables

File: /examples/demo-stringVariables.html

Source Code:


// String Variables

// Declare some variables
var tabInThisVariable = "This is a tab\tand this is after it";
var aNewLineInTheMiddle = "This line has a newline\nin its middle.";

// Declare a mystery variable
var whatWillThisBe = "This is a string".length;

// Display the contents of a variable followed by two blank lines
document.write(tabInThisVariable);
// Add a newline after the line above and then an extra blank line
document.write("\n\n");

// Display a variable with a \n in its middle
document.write(aNewLineInTheMiddle);
// Add a newline after the line above and then an extra blank line
document.write("\n\n");

// Display the mystery variable with text in front of it and
// <br> tags after it
document.write("The mystery variable is ");
document.write(whatWillThisBe);
document.write("<br><br>");

// Display the mystery variable with a concatenation wrapping with
// HTML paragraph tags!
document.write("<p>Here's it is again " + whatWillThisBe + "</p>");

Example Output:

Explanation:

  • Line 4 declares the variable "tabInThisVariable" which has an embedded tab character ("\t").
  • Line 5 declares the variable "aNewLineInTheMiddle" which has an embedded newline character in its middle.
  • Line 8 declares the mystery variable.
  • Line 11 prints the "tabInThisVariable" variable to the page. The tab character has a gap bigger than a space.
  • Line 13 prints two newline characters to the page.
  • Line 16 prints the "aNewLineInTheMiddle" variable to the page. The embedded newline character splits the display to two lines.
  • Line 18 prints two newline characters to the page.
  • Lines 22-24 prints the mystery variable to the page with text in front of it and after it, using HTML break tags instead of newline characters. This only works in web browsers though because the browser interprets the <br> tag and converts it to a \n.
  • Line 28 also prints the mystery variable to the page with text in front of it and after it. This time concatenation is used. We'll be using this technique a lot in the future!
