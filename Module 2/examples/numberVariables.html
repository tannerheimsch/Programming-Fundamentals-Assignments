Unit 2: JavaScript Syntax and Variables

Example: Number Variables

File: /examples/numberVariables.html

Source Code:


// Numbers in JavaScript
//  Declare variables and assign values at the same time
var count = 1;
var price = 12.99;
var priceString = "12.99"; // This is not a number!

//  Display variables
document.write("count: ");
document.write(count);
document.write("\n\n");

document.write("price: ");
document.write(price);
document.write("\n\n");

document.write("price + count: ");
document.write(price + count); //add two numbers together
document.write("\n\n");

document.write("priceString: ");
document.write(priceString);
document.write("\n\n");

document.write("priceString + count: ");
document.write(priceString + count); //add a number and a string together
document.write(" Huh?");
document.write("\n\n");

Example Output:

Explanation:

  • Lines 3-4 declare number variables. Note the lack of quotation marks.
  • Line 5 declares a string variable even though the contents look like a number. The double-quotes make it a string.
  • Lines 8 prints the "count" variable to the page. The write function converts the number to a string for display automatically.
  • Lines 12 prints the "price" variable to the page.
  • Lines 16 adds price and count together and prints the results to the page.
  • Lines 20 prints the "priceString" variable to the page. Since it is already a string the write function doesn't need to convert it.
  • Lines 24 "adds" the "priceString" variable to the "count" variable and prints the results to the page. However, the plus sign (+) in this case converts the number to a string and then combines both strings together. This is called concatenation.
