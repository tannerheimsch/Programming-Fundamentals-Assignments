Unit 2: JavaScript Syntax and Variables

Example: document.write()

File: /examples/documentWrite.html

Source Code:


// Print some text to the page
document.write("Printing some text to the page.");

// Print 2 newline characters to the page.
document.write("\n\n");

// Print more text to the page.
document.write("Printing some more text to the page.");

// Print 2 HTML <br> tags to the page.
document.write("<br><br>");

// Print a number to the page
document.write(123);

// Print 2 HTML <br> tags to the page.
document.write("<br><br>");

// Print a number with decimals to the page.
document.write(123.99);

// Print a string that's really long using concatenation.
document.write("Here's a really really really really really really really " +
               "long line of text that uses one document.write but " +
               "breaks the code into multiple lines.");


// Print a string that's really long using concatenation but use a variable.
var message = "Here's a really really really really really really really " +
              "long line of text that uses one document.write but " +
              "breaks the code into multiple lines.";

document.write(message);

Example Output:

Explanation:

  • Line 2 prints some text to the page.
  • Line 5 prints some newline characters to the page, causing a blank line before the next line of text prints.
  • Line 8 prints some more text to the page.
  • Line 11 prints two HTML <br> tags to the page - another way to get a blank line before the next line of text, but only in the web browser! The \n approach is universal across programming languages. Browsers don't understand \n characters unless you output the results in a <pre> tag which this code does.
  • Line 14 prints a number to the page. The "write" function converts the number to a string automatically. Be wary of this! This is converting a data type and may not give you what you expect.
  • Line 17 prints more <br> tags for another blank line.
  • Line 20 prints a number with decimal places to the page.
  • Line 23 prints a very long string using string concatenation to keep the line under 80 characters.
  • Line 28 prints a very long string that we place in a variable. This lets us separate our output from our processing.
