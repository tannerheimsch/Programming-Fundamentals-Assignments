Unit 2: JavaScript Syntax and Variables

Welcome to Unit 2 - JavaScript Syntax and Variables

Objectives:

  1. Describe and use the syntax that is used in JavaScript programming.
    • Remember that "syntax" refers to the language (words, spelling, grammar, etc) that JavaScript expects the program to provide.
  2. Describe and use variables, and understand the importance of data types.
