You can't just type these numbers in! You have to use the (three) variables and a little math along with your document.write() statements.

You also have to output some text, then a number, then some text. How do we do that? Here's a hint (well, its more than a hint). Here's the output example:

Fred is 25 years old.

And, here's the code we would use to produce that output example.


// Output on one line

var name = "Fred";
var age = 25;

document.write(name);
document.write(" is ");
document.write(age);
document.write(" years old.");

// You could also do it in one line of code.
document.write(name + " is " + age + " years old.");

So now, use this example and what you've already learned about variables to create your program.

Remember, you must use variables for the page counts. You must use math to compute the total. You must use variables to build up the output statement.
