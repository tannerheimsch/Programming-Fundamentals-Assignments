Type Casting

The prompt keyword lets us prompt for a value from the user. So if we write this code: 

var firstNumber = 0;
firstNumber = prompt("What is the first number?");

The user will see this:

It's a nice simple way to get data into a JavaScript program. And it's perfect for exploring the fundamentals of programs, including data types.

Data that comes into a computer program from keyboard input is string data. If you are making a calculator and you prompt for input, it will be a string, not a number.

The number 22 is completely different from the string "22". Especially to the computer.

To solve this problem, we use type casting. This is the process of converting data from one type to another. 

var firstNumber = 0;
var secondNumber = 0;
var total = 0;

firstNumber = prompt("What is the first number?");
secondNumber = prompt("What is the second number?");

total = firstNumber + secondNumber;

document.write(total);

The value that comes back from the prompt will be a string, not a number. If we run the program above, you'll get a very unexpected result. Try it out for yourself. Write up the program, run it, and enter two numbers. You'll find they have been concatenated instead of added!

So we have to convert the values from strings to numbers before we can use it.

We do that with the Number function. 

var firstNumber = 0;
var secondNumber = 0;
var total = 0;

firstNumber = prompt("What is the first number?");
secondNumber = prompt("What is the second number?");

// Added these two lines
firstNumber = Number(firstNumber); // convert from string to number
secondNumber = Number(secondNumber); // convert from string to number

total = firstNumber + secondNumber;

document.write(total);

The values are strings, but then we use the Number function to convert the values from strings to numbers, and we save the result of the conversion back to the original variable, overwriting its original value.

In some languages, you will need separate variables for the string and the number. You'll learn more about data types in a future lesson.

For now, just understand that data from the keyboard comes in as strings, and if you want to do math on it, you need to convert that data to numbers.
