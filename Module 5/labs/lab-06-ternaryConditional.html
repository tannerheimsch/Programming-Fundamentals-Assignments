Lab problem statement

Write a program that asks a user for a number greater than or equal to 100. If the user enters a number that is greater than or equal to 100 then output, "Thank you!". If the entered number is not greater than or equal to 100 then output, "That number is not correct, please try again."

This program must be implemented using a ternary, not an if..else statement.
