Lab problem statement

A program is required that asks a user to enter a number between 1 and 100 that does not include the end points (1 and 100). When a number in the correct range is entered then output the number and a message indicating success. If the number is not in the range then output the number and ask the user to try again.

For full points, your program should include only one output statement.
