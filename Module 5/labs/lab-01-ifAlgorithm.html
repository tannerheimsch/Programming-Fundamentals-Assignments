Problem Statement

A program is required that asks a user to enter a number greater than 10. If the number is not greater than 10, then alert the user that they did not enter a number greater than 10. In either case output the number to the page as shown in the example output at the bottom of the lab.

For full points, your program should include only one output statement.

//YOUR SOLUTION ALGORITHM STARTS AFTER THIS LINE

NumberChecker
  Initialize userNumber to 0
  Initialize output to ""

  Prompt for userNumber with "Please enter a number that is greater than 10:"
  Convert userNumber to number

  IF userNumber > 10 THEN
    output = "You entered " + userNumber + "."
  ENDIF
  IF userNumber <= 10 THEN
    output = "You entered " + userNumber + "." + "\n" + "This number is not greater than 10."
  ENDIF

  Display output
END
