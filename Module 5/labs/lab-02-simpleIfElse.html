Lab problem statement

Write a program that asks a user to enter a string. Then ask the user to enter a string that will sort lower alphabetically than the first string. (What does it mean when something sorts lower or higher? Z sorts lower than A, and A sorts higher than Z.) If the second string sorts lower alphabetically than the first then output a message that includes both strings and a success message. If the second string does not sort lower than the first string then output the strings and ask the user to try again.

Hint: Look at the ASCII Table. Remember that lower case letters will sort lower than upper-case letters too.

For full points, your program should include only one output statement.

StringSortCheck
  Initialize userString1 to ""
  Initialize userString2 to ""
  Initialize output to ""
  
  Prompt for userString1 with "Please enter any string:"
  Prompt for userString2 with "Please enter a string that sorts lower alphabetically than your previous string:""

  IF userString1 < userString2 THEN
    output = "Nice job! \"" + userString2 + "\" sorts lower than \"" + userString1 + "\"."
  ELSE
    output = "\"" + userString2 + "\" sorts higher than \"" + userString1 + "\", please try again."
  ENDIF

  Display output
END
