Unit 5: Making Decisions

Welcome to Learning Unit 5 - Making Decisions

In this learning unit, you'll revisit selection as you learn to incorporate decision logic into your programs. You'll learn how to handle simple one-off choices, binary choices, and then you'll learn how to handle software programs where multiple input values determine the program's logical flow.

While the concepts in this lab seem simple, they make programs exponentially harder to test and maintain. For every decision the computer has to make, the developer has to plan to test that part of the code.

But don't worry. In this unit you'll find lots of examples and advice that will prepare you for the assessments. Be sure to review them all and ask your instructor for clarification if you need it.
