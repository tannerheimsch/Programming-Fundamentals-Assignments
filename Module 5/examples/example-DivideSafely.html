Example: Divide Safely

File: examples/example-DivideSafely.html

Problem Statement

A program is required that asks a user to enter two numbers. If the second number is not zero then divide the first number by the second number and output the results to the page. If the second is zero then alert the user that they entered a zero and should try again. Do not output anything to the page if the second number is zero.

5. JavaScript Output

Write the JavaScript code based on everything you've done.

Here's the code:


// Safely Divide Two Numbers

function SafelyDivideTwoNumbers() {
  // Declare variables
  var numberOne;
  var numberTwo;
  var output;
  var results;

  // Get input from user
  numberOne = prompt("Please enter a number");
  numberTwo = prompt("Please enter another number");

  // Convert input strings to numbers
  numberOne = Number(numberOne);
  numberTwo = Number(numberTwo);

  if (numberTwo === 0) {
    output = "You have entered a zero. Please try again.";
  } else {
    results = numberOne / numberTwo;
    output = numberOne + " divided by " + numberTwo + " is " + results;
  }

  document.write(output);
}

// Run the program
SafelyDivideTwoNumbers();
