Problem Statement

Design a program that will prompt a user for three characters, accept those characters as input, sort them into ascending sequence and output them to the screen.

1. Nouns and Verbs

Identify the nouns and verbs in the problem statement.

PROBLEM STATEMENT:
  Design a program that will prompt a user for three characters,
  accept those characters as input, sort them into ascending
  sequence and output them to the screen.

Nouns: first character, second character, third character

Verbs: prompt, accept, sort, output

2. Defining Diagram

Convert the nouns and verbs into variable names and processes and create a defining diagram.

INPUTS:
  character1
  character2
  character3

PROCESSING:
  prompt for characters
  accept three characters
  sort three characters
  output three characters

OUTPUTS:
  character1 unsorted
  character2 unsorted
  character3 unsorted

  character1 sorted
  character2 sorted
  character3 sorted

3. Solution Algorithm

Create a solution algorithm based on the defining diagram.

SortThreeCharacters
  Initialize swap to ""
  Prompt for character1 with "Enter a character"

END

function SortThreeCharacters(){

  // Read Three Characters
  // Declare variables
  var character1;
  var character2;  // not really alphabetical, but ok logically.
  var character3;
  var output = ""
  var swap;

  // Prompt user for characters and store in variables
  character1 = prompt("Enter a character");
  character2 = prompt("Enter a character");
  character3 = prompt("Enter a character");

  //Display the entered characters

  output += "<h3>Entered Characters</h3>";
  output += "<p>Character One:\t\t" + character1 + "</p>";
  output += "<p>Character Two:\t\t" + character2 + "</p>";
  output += "<p>Character Three:\t\t" + character3 + "</p>";

  // Sort the characters
  if (character1 > character2) {
    swap = character1;
    character1 = character2;
    character2 = swap;
  }

  if (character2 > character3) {
    swap = character2;
    character2 = character3;
    character3 = swap;
  }

  if (character1 > character2) {
    swap = character1;
    character1 = character2;
    character2 = swap;
  }

  // Display the sorted characters
  output += "<h3>Sorted Characters</h3>";
  output += "<p>Character One:\t\t" + character1 + "</p>";
  output += "<p>Character Two:\t\t" + character2 + "</p>";
  output += "<p>Character Three:\t\t" + character3 + "</p>";

  document.write(output);
}

// run program
SortThreeCharacters();
